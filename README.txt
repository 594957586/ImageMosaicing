Please ensure that the following libraries are installed using pip3 on your machine:
joblib, matplotlib, numpy,opencv-contrib-python, opencv-python,scipy, shapely.
All of them can be installed using "pip3 install"




To run the application please place the imgaes in the "/images" directory(It contains samples already).
Run the application with the following command:
	python3 main.py [Image1] [Image2] [Number of iterations]
For example:
	python3 main.py edinburgh1.png edinburgh2.png 100





The output will be saved in this directory as output.jpg
